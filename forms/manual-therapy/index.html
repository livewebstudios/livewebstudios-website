Adding & Updating Events on Your Website

A short walkthrough for Manual Therapy Consultants — how to add a new event, edit an existing one, or take one down. About five minutes once you've done it a couple of times.

Hi Alex,

Here's the walkthrough we talked about. It's the same method I use when I add a new seminar to your site: duplicate an existing event, then swap in the new details.

Two things to keep in mind before you start:

  • Always duplicate an item marked "In stock." Never a Draft.
  • You're working on a copy, so the original event stays untouched. If anything looks off partway through, just delete the new draft and start over.
1

Start with a copy

Go to Products → All Products. Find an event similar to the one you're adding. Hover over it, then click Duplicate. Make sure the item you copy says "In stock," not "Draft."

2

Rename it

Your new copy opens in the editor with "(Copy)" added to the name. Click into the title field and rename it for the new event.

3

Update the date and location

Scroll down to Product short description and Product long description. Click "Code" instead of "Visual" for both boxes. I know it looks like code — it's actually the safer, easier way to change just the date and location without messing up the formatting. Update the text in both boxes to match the new event.

4

Set the sort order

Click Advanced on the left side of the Product data box. In Menu order, type the event date as YYMMDD. September 26, 2026 becomes 260926. This is what keeps your events sorted by date automatically.

5

Open the product image

On the right side of the screen, find Product image and click the picture.

6

Choose a new photo

A media library window opens. Look for an image with a name that matches the event's city or theme. Click it, then click "Set product image" in the bottom right.

7

Publish

Once everything looks right, click Publish in the top right. That's it — the event is live.

Editing or removing an event

Fixing something on an event that's already posted? Skip the duplicate step. From the Products page, click Edit next to that event, make your changes, and click Update.

Taking an old event down? From the same Products page, click Trash next to it. It comes off the site right away — and it's not gone for good, you can restore it from the Trash tab if you ever need it back.

Call or text anytime if you get stuck partway through. Happy to walk through it live.
